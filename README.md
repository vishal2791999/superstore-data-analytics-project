# Superstore Data Analytics Project

## Project Overview

This project demonstrates a complete data analytics workflow using the Superstore dataset.

The project includes:

* Data Cleaning
* Exploratory Data Analysis (EDA)
* SQL Business Queries
* KPI Identification
* Dashboard Creation
* Business Insight Reporting

## Tools Used

* Microsoft Excel
* SQL
* Power BI

## Dataset

The Superstore dataset contains retail sales data including:

* Orders
* Customers
* Products
* Regions
* Sales and Profit

## Key KPIs

* Total Sales
* Total Profit
* Total Orders
* Average Order Value
* Profit Margin

## Insights

* Technology category generates the highest revenue.
* West region generates the highest profit.
* Some furniture products have high sales but low profit due to discounts.
